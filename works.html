架空コーポレートサイト制作　※ 現在制作中

株式会社ハートフルメンテ（架空の企業様）　制作時期：2020.10

■ 企画 / HTML / CSS / JavaScript

飲食・小売店舗の設備メンテナンス業の顧客様を想定し、 架空のコーポレートサイトを制作中です。

サイト制作の目的を「名刺代わり・企業イメージ向上・顧客獲得・求人募集」の4つと 定義して設計しております。

なお、こちらをベースに企業様サイトのご依頼を受けた際は スピーディな制作対応が可能となります。

見積りフォームにおける自動計算機能・割引機能等を実装

某刺繍加工業 企業様サイト（リンク公開不可）　制作時期：2020.09

■ HTML / CSS / JavaScript

通販のお見積りフォームにおける自動計算機能・割引機能および入力制限機能（バリデーション）を実装致しました。

割引を適用する対象範囲など、お客様のご要望を細かくヒアリングした上で機能追加をしています。

エンドユーザー様に料金体制の意図が伝わりやすいよう、 表示するテキストや見せ方も工夫して制作し好評を得られました。

簡易ブラウザアプリ制作

某企業様（リンク公開不可）　制作時期：2020.06～07

■ 企画 / Python / Bottle / bootstrap

勤務先の業務で使用する、過去案件検索用のアプリを制作致しました。

社内サーバ内にあるExcel（案件の進捗一覧）についてバッチ実行によりデータをアップロードし、 外出先からでもスマホで内容を検索・確認できるよう実装しています。

その結果、確認のための出社や出張先から社内への電話確認が不要／ 急な問い合わせにもすぐ対応できるようになったと好評を得られました。
